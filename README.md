## 📋 Overview

The **AI Financial Dashboard** is a personal finance tracking and analytics app that allows users to visualize, analyze, and understand every financial transaction through interactive charts and AI-assisted insights. Built as a production-oriented data app using Streamlit.

---

## ✨ Features

- 💰 Track income, expenses, and savings over time
- 📈 Interactive charts and KPI cards for financial health overview
- 🤖 AI-assisted transaction categorization and reporting
- 🔍 Filter and drill down into specific time periods or categories
- 📊 Summary analytics: monthly trends, category breakdowns, net flow
- 📂 Upload and manage your own transaction data (CSV/Excel)

## 📲 Setup & Run Locally

```bash
# Clone the repository
git clone https://github.com/Vinodvinum/Financial_dashboard.git
cd Financial_dashboard

# Install dependencies
pip install -r requirements.txt

# Run the Streamlit app
streamlit run app.py
```
